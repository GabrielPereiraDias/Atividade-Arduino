# 💡 Desenvolvimento de Sistemas: Atividade Arduino no Tinkercad 🔌

![Status](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen?style=for-the-badge)

Esta atividade foi desenvolvida no curso **Técnico em Desenvolvimento de Sistemas** utilizando o **Tinkercad** para simulação de circuitos com Arduino.

O objetivo da atividade foi praticar conceitos básicos de eletrônica e programação, realizando testes com LEDs e componentes digitais através da plataforma de simulação.

---

## 📂 Atividades Desenvolvidas

| Atividade | Descrição | Status |
| :--- | :--- | :--- |
| `LED Arduino` | Montagem e programação de circuitos simples utilizando LEDs no Arduino através do Tinkercad. | ✅ |
| `Simulação Digital` | Testes de funcionamento dos componentes eletrônicos em ambiente virtual. | ✅ |
| `Programação Arduino` | Desenvolvimento de códigos básicos utilizando a linguagem do Arduino (C/C++). | ✅ |

---

## ⚠️ Observações

> Todos os testes foram realizados no ambiente virtual do Tinkercad.

> A atividade teve foco no aprendizado da integração entre programação e componentes eletrônicos básicos.

---

## 🛠️ Ferramentas Utilizadas

* **Tinkercad:** Simulação dos circuitos eletrônicos.
* **Arduino UNO:** Plataforma utilizada na atividade.
* **Linguagem C/C++:** Programação dos componentes eletrônicos.
